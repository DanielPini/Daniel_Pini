CoMA Ecommerce Rebuild

Template refactor, database cleanup, and checkout UX improvements for a WordPress commerce site.

Goal

Improve performance, reduce plugin bloat, and streamline purchase funnel.

Features

  • Custom PHP templates replacing heavy page builders.
  • Query optimization & index review in MySQL.
  • Checkout field reduction & validation feedback.

Architecture

functions.php → enqueue minimal assets
custom-template.php → lean markup
DB audit → remove orphan meta
perf hooks → transient caching

Challenges & Solutions

  • Legacy plugin conflicts → staged deactivation + replacement.
  • Slow product queries → added composite indexes.
  • Checkout abandonment → simplified steps & clearer errors.

Performance

Metric Before After
Time To First Byte 850 ms 420 ms
Checkout Conversion 62% 74%
Asset Count 143 68

Outcome

Faster browsing, higher conversion, easier maintenance footprint.
