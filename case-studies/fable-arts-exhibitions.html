FABLE ARTS Digital Exhibitions

Framework powering interactive artist exhibitions (2020–2025): modular layout, scalable media, narrative progression.

Scope

  • Template system for varying exhibition formats.
  • Media preloading & adaptive image sets.
  • Curator-friendly configuration JSON.

Architecture

config.json → loader
build DOM modules (hero, gallery, narrative)
attach progressive enhancement (transitions, captions)
analytics hooks (engagement)

Performance Improvements

Reduced median image payload by switching to WebP + deferred non-critical sections.

Outcome

Consistent launch process for new exhibitions; reduced setup time for each artist onboarding.
