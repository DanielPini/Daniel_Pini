Stephen Fleri Artist Portfolio

Responsive gallery experience with dynamic filtering, accessible structure, and optimized media delivery.

Goal

Create a visually clean, fast-loading portfolio enabling users to filter works by series and medium.

Features

  • React enhancement layer for client-side filtering.
  • Lazy-loaded image sets with intrinsic aspect ratio placeholders.
  • Keyboard and screen reader accessible gallery navigation.

Architecture

index.html (semantic sections)
React mount → #gallery-root
fetch works.json → normalize categories
filter state → render subset (virtual list ready)
IntersectionObserver → image src swap

Challenges & Solutions

  • Large image payload → generated multiple sizes & used srcset.
  • Filtering jank → minimal state & memoized map.
  • A11y alt coverage → curated alt text per artwork.

Performance

Metric Before After
Largest Image Payload 12.4 MB 4.1 MB
First Meaningful Paint 2.9 s 1.5 s
Accessibility Score 82 100

Outcome

Improved engagement and reduced bounce with fast, filterable viewing experience.
