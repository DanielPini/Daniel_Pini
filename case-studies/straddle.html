Straddle Poker Blinds Timer

Accurate tournament pacing tool with configurable blind levels and audible transitions.

Goal

Provide a simple browser-based timer to manage blinds progression without manual recalculation.

Features

  • Configurable blinds schedule JSON.
  • High-precision interval loop (drift correction).
  • Audio alert and visual flash on level change.

Architecture

schedule.json → parse levels
Timer → setTimeout drift adjust
UI update → remaining time & next level
Audio cue → play on transition

Challenges & Solutions

  • setInterval drift → timestamp difference correction.
  • User pause/resume → persisted state snapshot.
  • Mobile viewport scaling → fluid layout units.

Performance

Minimal CPU footprint; updates limited to once per second except on transition frames.

Outcome

Reliable pacing aid adopted for casual tournament sessions.
